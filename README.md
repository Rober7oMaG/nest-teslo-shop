# Teslo Shop API
1. Clone the repository
2. Execute
```
yarn install
```
3. Have Nest CLI installed
```
npm i -g @nestjs/cli
```
4. Clone .env.template file and rename it to .env
5. Fill .env file with your data
6. Start database
```
docker-compose up -d
```
7. Start app on development mode
```
yarn start:dev
```
8. Execute seed
```
http://localhost:3000/api/seed
```


## Used Stack
* NestJS
* PostreSQL
